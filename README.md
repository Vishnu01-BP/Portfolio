# Portfolio Project

This repository contains the portfolio website for Vishnu B P.

## Structure

- `client/` - Next.js frontend application
- `resume__general.pdf` - General resume
- `resume_internship.pdf` - Internship-focused resume

## Quick Start

```bash
cd client
npm install
npm run dev
```

For detailed instructions, see the [client README](./client/README.md).

## Deployment

The portfolio is designed to be deployed on Vercel (free tier).

See the client folder for complete setup and deployment instructions.
