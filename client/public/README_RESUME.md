# Add Your Resume PDF Here

## Steps to Add Resume:

1. **Generate PDF from LaTeX:**
   - Go to https://www.overleaf.com/
   - Create new project
   - Paste your LaTeX resume code
   - Compile and download PDF

2. **Place PDF in this folder:**
   ```
   client/public/Vishnu_BP_Resume.pdf
   ```

3. **The download button will work automatically!**
